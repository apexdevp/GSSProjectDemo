Version Control Demo

Hello, Git.

A tiny two-page site you can use to practice the everyday Git workflow — commit, diff, branch, and merge.

Home About

1 · Start tracking these files

Put both HTML files in a folder, then turn it into a repository:

# initialise and make the first commit
git init
git add index.html about.html
git commit -m "Add home and about pages"

2 · Make a change and see the diff

Edit the headline above (try changing "Hello, Git." to your name), then:

git status
git diff
git commit -am "Update homepage headline"

3 · Branch, then merge

Try a change in isolation before bringing it back to main:

git switch -c new-feature
# ...edit a file, then commit...
git switch main
git merge new-feature
